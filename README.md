# Alma Media - Agencia de Marketing Digital

Sitio web Next.js 14 para Alma Media, agencia de marketing digital en la Región del Biobío.

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL con Prisma ORM
- **Auth:** NextAuth.js
- **Animations:** Particles.js

## 🛠️ Instalación Local

```bash
npm install
npm run dev
```

## 🌐 Deploy en Coolify

1. Push a GitHub
2. Conectar repo en Coolify
3. Build command: `npm run build`
4. Start command: `npm start`
5. Port: `3000`

Hecho con ❤️ en Biobío
